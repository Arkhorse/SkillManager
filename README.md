# Features
<!--
This should be a bullet point list of everything the mod does
-->
This mod allows you to set a multiplier on experience gained whenever you gain any skills. The settings allow you to set the mult for each individual skill

# Install Instructions
<!--
This should be a bullet point list of everything the user must do in order to use the mod, including installing ML
-->
Download MelonLoader, install the current supported version (see the top right of the TLD Modlist), install ModSettings then install this mod. All required mods (and this one) go into the MODS folder.
